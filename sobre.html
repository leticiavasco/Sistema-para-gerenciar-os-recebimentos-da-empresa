Sobre o Sistema

O Sistema de Contas a Receber foi desenvolvido para auxiliar pequenas e médias empresas no controle de faturas, vencimentos e recebimentos. Ele oferece uma interface simples, intuitiva e eficiente para que o usuário gerencie seus clientes e acompanhe todos os pagamentos pendentes ou concluídos.

O que o sistema faz?

  • Registra faturas com informações completas: cliente, número, vencimento, valor e status.
  • Permite acompanhar pagamentos e manter histórico organizado.
  • Salva automaticamente as faturas no navegador (usando LocalStorage), mantendo o cadastro mesmo após fechar a página.
  • Possui interface responsiva, adaptada para celulares, tablets e computadores.
  • Permite excluir faturas de forma simples e rápida.

Por que criamos este sistema?

A organização financeira é uma parte essencial de qualquer empresa, mas muitos negócios ainda utilizam planilhas manuais ou métodos desorganizados para lidar com contas a receber. Nosso sistema foi criado com o objetivo de:

  • Facilitar o controle financeiro;
  • Reduzir erros e atrasos de pagamento;
  • Oferecer uma ferramenta simples e acessível para estudantes e empresas iniciantes;
  • Demonstrar conceitos de desenvolvimento web conforme os critérios de trabalho acadêmico.
